# react-native-skia-kit

A Flutter-like UI Kit for React Native that renders all UI components on a single Skia Canvas, providing high-performance custom widgets with Yoga layout, gesture handling, and theme support.

## Installation


```sh
npm install react-native-skia-kit react-native-nitro-modules

> `react-native-nitro-modules` is required as this library relies on [Nitro Modules](https://nitro.margelo.com/).
```


## Usage


```js
import { multiply } from 'react-native-skia-kit';

// ...

const result = multiply(3, 7);
```
